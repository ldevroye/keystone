## Report

This folder is destinated for the report making of the Master Thesis. It is done using LaTeX.

## Usage

Working on it was used using *latex-workshop* and *Latexmk: This is Latexmk, John Collins, 31 Jan. 2024. Version 4.83*

For a proper compiling using it. Import the **[settings.json](settings.json)** file in *'keystone/.vscode/settings.json'*.  
The auxiliary files are put in the *'build/output/'* and *main.pdf* is at the root of the subfolder (*'report/'*).

### Makefile.  

<code>$make</code> to compile  

<code>$make clean</code> to clean the auxiliaries and pdf
